# 🎬 Gestor de Filmes (CLI)

Sistema em Python para gerenciamento de filmes com persistência em JSON.

## 🚀 Funcionalidades
- Adicionar filmes com nome, nota e status
- Listar filmes
- Editar filmes
- Buscar filmes
- Estatísticas (quantidade e média de notas)

## 💻 Tecnologias
- Python
- JSON

## ▶️ Como executar

python filmes.py
