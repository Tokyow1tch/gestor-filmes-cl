 # 🎬 Gestor de Filmes (CLI)

Sistema em Python para gerenciar filmes com persistência em JSON.

## 🚀 Funcionalidades
- Adicionar filmes com nome, gênero, nota e status
- Listar filmes
- Editar filmes
- Buscar filmes
- Estatísticas (quantidade, média de notas)

## 💻 Tecnologias
- Python
- JSON

## ▶️ Como executar

python gestor_filmes.py
